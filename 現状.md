
#　状況
- M1 MacBook Pro（以下 Mac）：ホスト機体（開発環境）
- M1 MacBook Air（以下 Air）：Asahi Linux インストール済み。最終実行用機体

# 実装方針
Mac ホスト上の **UTM** で Fedora 43 ARM64 VM を構成する。
VM 上で梵天を開発・テスト。
最終的には Air 実機上で梵天を実行。

# v0.0
Asahi Linuxのベース（カーネル + Mesa + Honeykrisp GPU ドライバ）は必須
デスクトップ環境（GNOME/KDE等）は削除
systemd は最小化設定
wgpu が DRM/KMS 直接叩くので、X11/Wayland不要
